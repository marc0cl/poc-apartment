# poc-apartment

Repositorio **docs-first** para construir una plataforma multi-tenant para corredoras de propiedades (portal público + portal admin).

Este repo está organizado para permitir desarrollo **asistido/ejecutado por IA** con mínimo riesgo de alucinaciones.

---

## 1) Qué leer primero (ley del repo)

### Normativa de IA (OBLIGATORIO)
- Protocolo: `docs/ai/protocolo.md`
- Plantilla de respuesta: `docs/ai/response_template.md`
- Memoria del proyecto: `docs/ai/memoria_proyecto.md`
- Quality Gate: `docs/ai/quality_gate.md`

### Decisiones
- Plantilla ADR: `docs/adr/template_adr.md`

### Historia / Iteraciones
- `docs/iterations/`

---

## 2) Regla operativa (para humanos y agentes IA)

- No se inventan endpoints/modelos/stack/comportamientos.
- Toda entrega debe listar **Fuentes consultadas** (paths del repo).
- Si una decisión cambia arquitectura, seguridad, datos o invariantes del producto: requiere **ADR**.
